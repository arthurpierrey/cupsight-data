# Cuppanion data

Base de torréfacteurs specialty utilisée par l'app **Cuppanion** (scan de paquets de café par IA).

`roasters.json` : nom canonique, alias, pays/ville, style de torréfaction et notes pour ~110 torréfacteurs (France, Europe, incontournables monde). L'app la télécharge automatiquement une fois par mois.

## Styles

- `ultralight` — ultra clair / nordique extrême (SEY, Glitch, Manhattan…)
- `light` — clair
- `modern` — specialty équilibré
- `classic` — traditionnel

## Contribuer

Un torréfacteur manque ou est mal classé ? Ouvre une PR sur `roasters.json` (et incrémente `version`).
